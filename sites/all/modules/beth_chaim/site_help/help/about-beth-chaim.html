Beth Chaim Web Site

The Beth Chaim web site is built on the Drupal Content Management system. Drupal is an extremely flexible and scalable content management package that provide the user with full control over the development of a website

It is very useful for community group web sites that have a variety of users that require different levels of access or contol of the web site's content.

It also has a very large library of ready to use modules that easy plug into the system to expand functionality in virtually limitless configuration possibilities.
